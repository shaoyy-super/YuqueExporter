已经确定的玩法基础：玩家在每个地图中从0开始“建设”5个地标建筑，使其从一片空地变为高楼大厦（0级~6级，共7档）之后视为完成这个地图的目标，去往下一个地图。

角色定位：富有正义感的见习怪盗

参考：女神异闻录5，猫眼三姐妹，怪盗基德

剧情：每个平行世界都有自己的麻烦，【布鲁克林街区的偷盗案层出不穷】，【蜘蛛侠和章鱼博士大打出手】，【牛顿丢失了苹果导致写不出万有引力定律】，玩家扮演的主角作为宇宙怪盗事务所的【见习怪盗】，需要穿梭于不同的平行宇宙，解决一个接一个的麻烦，不断提升实力，积累财富，达成自己的目标。
