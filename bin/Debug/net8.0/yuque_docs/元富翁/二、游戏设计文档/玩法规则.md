## 设计思路
+ 三国杀的阵营规则+金铲铲-海克斯型的选牌规则+模拟城市建设的包装。
+ 单局内获得随机身份，根据阵营选择合适的国家，根据阵营、国家特色、其他玩家策略，决定自己的局内策略并争取满足阵营胜利条件。
+ 与游戏主玩法相辅相成，游玩主玩法解锁阵营、建筑，游玩该玩法来获取主玩法的奖励和资源。

# 功能说明
## 名词说明
### 基础属性
#### **属性类型**
三大资源属性：食物、人口、科技

货币属性：金钱

#### **属性获取**
+ 玩家通过建造建筑来获取或消耗资源，通过攻防获得或消耗资源。
+ 金钱通过建造商业类建筑获取。

#### **属性作用**
+ 食物、人口、科技作为基础资源，当结算时任一属性低于0，则会扣血，扣血之后该资源恢复到国家特定值。
+ 金钱作为进攻资源，通过消耗金钱来启动武器进行进攻，或者商店购买攻防卡牌。

### 阵营
#### 阵营划分
+ 类三国杀的阵营包装，对应主公、忠臣、反贼、内奸，分别为联盟中心、盟国、敌国、叛乱国。

#### 获胜条件
+ 联盟中心/盟国：击败阵营外的其他玩家
+ 敌国：攻破联盟中心
+ 叛乱国：先击败联盟中心外所有人，再攻破联盟中心

#### 获取时机
+ 开始对局后系统会随机选择身份阵营指派给玩家。

### 国家
#### 概述
+ 每个玩家进入对局之后选择国家。
+ 每个国家有其对应的专属技能或者buff。

#### **类型**
包含多种类型，提供不同的策略，例如攻击型、资源型等。

#### 解锁条件
+ 通过游玩建筑系统解锁，完成建筑系统中的城市后，进行国家解锁。
+ 通过商店购买、段位解锁等，具体依照后续设计。

### 国运
#### **概述**
+ 国运为生命值的包装，不同国家有不同的血量包装。
+ 在此基础上，联盟中心国运再+1，其他阵营不变。

#### **规则**
+ 基础属性值被攻击至负数时，扣除一点生命值，该基础属性随后恢复至特定值。
+ 生命值归零则玩家亡国判负。
+ 游戏中存在其他增加或减少国运的情况，具体依照策略设计。

### 建筑建造
#### 建筑卡牌获取
+ 选牌阶段每回合五选一，五张中3张对应3种基础属性的卡牌，2张效果建筑卡牌，连续选择3次，每回合共选择3张卡牌。

**建筑卡牌刷出规则**

+ 每张卡牌有对应的刷出条件和刷出权重，当前回合筛选出所有满足刷出条件的卡牌，按照权重，在满足上述（3+2）的条件下进行刷出。

#### 建造建筑
+ 玩家通过在建造阶段依照选择的卡牌来建造建筑。
+ 玩家快速选择好3个建筑后，随即场景中进行建筑建造。
+ 当选择建筑卡牌时，若场地中有空位，则进行建造，若没有，则只叠加属性。

#### 建筑类型
发育建筑：持续获得资源，例如本回合开始，每回合获得5点食物。

速攻建筑：立即获得资源，例如本回合立即获得15点食物。

效果建筑：提供额外效果，例如敌方攻击时，降低5点伤害。

+ 具体建筑类型和效果依照设计。

#### 玩法内建筑来源
+ 依照建筑系统进行解锁，包含单独解锁的建筑。
+ 依照国家进行解锁，解锁特定国家后解锁特定建筑，其他解锁条件依照后续设计。

### 商店
#### 商店内容
+ 不同种类、等级的攻击牌和防御牌。

#### 购买规则
+ 花费金钱进行购买。
+ <font style="color:#000000;">整个回合都可自由购买。</font>
+ 是否设置上限后续待定，若存在上限需要支持卡牌替换或出售。

#### 使用规则
+ 攻击卡牌购买后作为一种武器进入玩家武器库中。
+ 防御卡牌在被攻击后自动使用。

### 攻防
#### 攻防方式
+ 每个玩家默认存在多种基础攻击武器，选择不同资源类型，花费金钱可以直接攻击，并且可以调整倍数。
+ 玩家可以进行武器切换，不同武器有不同的伤害和效果，花费的金钱不同。
+ 购买的攻击卡牌也会作为武器，供玩家选择。

#### **攻防卡牌**
+ 攻防卡牌主要包含特殊的攻击手段和应对不同攻击的防御手段。
+ 攻击卡牌中的攻击手段不可调整倍数。
+ 攻击卡牌主动使用，防御卡牌被动使用。

#### 攻击
+ 玩家在攻击阶段选择对应的武器，选择对应的玩家以及对应的资源，进行攻击，每回合最多攻击两次。
+ 选择武器和选择玩家没有先后之分，可以即时进行。
+ 攻击时播放攻击动画，并播放对应资源的建筑损毁的动画表现。
+ 按照攻击对应的资源量来计算建筑损毁的效果和范围，可以结合具体的破坏形式进行设计。

#### 防御
+ 被攻击时被动使用防御卡牌，若有多张优先自动使用和攻击资源类型相同，数值相近的。

#### 伤害计算
+ 玩家每次攻击后，对该玩家和攻击目标进行结算。
+ 若对方没有防御卡牌，则对被攻击的资源直接扣除攻击对应的数量。
+ 若存在防御手段，则依照防御类型进行计算，伤害为扣除防御值后的数量。
+ 结算时任意资源变为负数，则扣除一点血，随后该资源恢复至特定值。

       特定值可由国家、阵营、建筑效果、回合数影响。

## 玩法系统规则
+ 为常驻玩法，满足解锁条件即可开启。

### 赛季和排行
+ 设置赛季制，每个赛季按当前赛季积分开启全服排行榜，排行榜容量：200

**积分及段位继承规则**

+ 新赛季的初始段位由上赛季的阶段段位决定，初始积分归零

### 积分和段位
+ 设立玩家积分和段位来展示玩家的水平
+ 玩家每完成一场对局，依照对局结果进行积分增加或者下降，达到段位分数线，来判断升段或者降段
+ 低段位玩家失败不掉分

:::warning
**玩家单局得分计算公式：（获胜得分和失败得分单独计算）**

单局得分 = 基础权重*（权重1 * 攻击成功次数 + 权重2 * 防御成功次数 + 权重3 * 建造建筑数） + 所属阵营获胜权重 * （权重4 * 攻击敌对阵营成功次数 + 权重5 * 击杀敌对阵营人数） + 修正值

:::

名词解释：

+ 敌对阵营：三方阵营，另外两方阵营均为自身敌对阵营

### 匹配
+ <font style="color:#000000;">优先匹配当前段位的玩家，若匹配时间>30s，则扩大匹配池至上下相邻两个大段位</font>
    - <font style="color:#000000;">如玩家为黄金Ⅱ段位，则扩大后的匹配范围是（白银Ⅰ、Ⅱ、Ⅲ和黄金Ⅰ、Ⅱ、Ⅲ和铂金Ⅰ、Ⅱ、Ⅲ）</font>
+ <font style="color:#000000;">根据段位设置最大匹配时间，若超过最大匹配时间未匹配到玩家，则填充机器人开始对局</font>
+ <font style="color:#000000;">高段位场不填充机器人</font>

### 奖励
#### 对局奖励
+ 每场对局胜利获得积分、金钱、道具奖励
+ 对局结束后直接发放奖励

#### 每日奖励
+ 设置多个每日目标，达成后可以获得道具和资源奖励。
+ 目标通常为完成X局对局，胜利X场，获得X点积分等。

#### 赛季奖励：段位奖励+排行奖励
+ 赛季结束时发放当前赛季段位奖励
+ 赛季结束时对全服排行榜的上榜玩家发放排行奖励
+ 奖励通过邮件发放

### 机器人及托管AI
+ 机器人根据行为逻辑设计区分多种能力的，不同段位填充不同类型机器人
+ 当玩家在对局中操作环节15s内无操作行为时，进入托管状态
    - 对局未结束时玩家回到对局，结束托管
+ 托管时，对应玩家头像旁显示托管状态，结束托管时，托管状态消失

## 多人玩法规则说明
![画板](https://cdn.nlark.com/yuque/0/2024/jpeg/43554293/1724063154916-1d102e25-c865-4989-9997-7d1cbbe4b623.jpeg)

+ <font style="color:#000000;">所有玩家准备就绪后，开始游戏。</font>
+ <font style="color:#000000;">设置准备就绪倒计时，倒计时结束后，回合开始。</font>

### 准备阶段
+ 系统随机给玩家指派一个阵营。
+ <font style="color:#000000;">玩家进行国家的选择，联盟中心玩家先选，其他玩家后选</font>
+ <font style="color:#000000;">先选+候选设置10+10秒选择时间，任意阶段时间结束后，若未选择则随机选择1个。</font>

### 回合开始
+ 单个回合分为建造阶段（卡牌选择阶段+建筑建造阶段）→攻防阶段，以及贯穿整个回合的购买阶段。

### 建造阶段
#### 卡牌选择阶段时间
+ 卡牌选择阶段为所有玩家同时进行，暂定共计15秒。

#### 获取卡牌
+ 开始回合后，依照卡牌选择规则进行选择。
+ 若未及时选择卡牌，则时间结束后默认全部选择第1张。

#### 建造建筑和结算
+ 本轮选择好所有卡牌之后，系统会统一在场景的对应区域进行建筑建设和资源结算。
+ 阶段时长固定，由建筑建造动画和资源结算时长所决定，结束后进入下一阶段。

### 攻防阶段
#### 行动顺序
+ 联盟中心先行动，其他玩家以联盟中心顺时针旋转后行动。
+ 每个玩家依次进行攻击行为。

#### 行动时间
+ 每个玩家给予一定时间作为攻击阶段，暂定10秒。
+ 每个玩家自身阶段最多可以攻击2次。
+ 若不攻击或者选择较快可以选择立即结束选择阶段，直接进行结算。

**攻击方式**

+ **方案1：**我方场景内选择目标和目标资源，再切换场景进行攻击目标的演出。

                   若有多次攻击需要全部选择好目标。

+ **方案2：**先选择玩家，进入目标玩家场景后，再选择目标建筑区域，进行攻击演出。

                   若有多次则按照先后顺序进入目标玩家场景进行攻击。

+ 任意方案只要没有做出攻击宣言，则可以即时切换目标玩家。攻击宣言指选定目标资源类型。

#### 攻防结算
+ 攻击他人时，每次攻击进行一次结算，结算资源量和玩家血量。
+ 玩家资源量=总资源量+防御效果-敌方攻击扣除量。若低于0则进行扣血，之后再回复至特定值。

#### 胜负结算
+ 当任意玩家血量≤0，则该玩家出局。
+ 若满足阵营胜负条件，则游戏结束。

### 商店购买
#### 购买规则
+ 整个回合都可购买攻防卡牌。

#### **使用规则**
+ 攻击卡牌主动使用，防御卡牌被动使用。
+ 除最终回合结算外，都可使用。

### 玩家行为
+ 玩家可以在任何阶段点击玩家信息区域前往对应玩家的场景查看，此时顶部资源条展示的资源量为该玩家的数量。

### 玩家战败后
+ 玩家战败后默认为观战视角，视角切换为当前主公国家，可以选择切换其他国家查看。
+ 玩家需要主动选择退出当前对局，若未退出且当前对局未结束，则断线重连后仍然处于该局对局中。

## 单人玩法规则说明
+ 为周期性活动，满足活动开启条件开启。
+ **具体以PVE玩法文档内为基础。**

### 规则说明
+ 沿用多人对战的玩法规则，但没有了阵营的概念，玩家只需要选择国家，进行1v1的闯关类PvE。
+ 我方挑战一个关卡后，无论胜负，会保留当前建造进度。
+ 关卡设置有回合数限制，并可设置一些特殊通关条件。

### 挑战次数
+ 每日设置一定数量挑战次数，达到次数后若想继续游玩，则需要购买挑战券。
+ 挑战券每日可以购买一定次数，同时也会投放在各类付费商品中。

### 奖励
+ 每关设置固定的骰子、金钱、贴纸等奖励。

#### 整体界面布局

+ 分配好阵营之后，提供国家选择阶段。
+ 展示当前拥有的所有国家，选中的国家展示国家详情，包含血量、技能或特殊buff。
+ 选择时间结束后，若未选择，则随机选择一个。

![](https://cdn.nlark.com/yuque/0/2024/webp/43554293/1722928543367-dc52ed97-b948-4175-bb54-fd446f6c7e54.webp)

#### 选牌阶段
+ 中间显示可选的牌，为了不挡住建筑建造表现，可以将选牌选择区域下移一些。
+ 选择时当前玩家拥有的攻防卡牌下移做一个隐藏，主要显示当前的建筑选牌。

**方案1：**边选牌边进行建筑建造，每次选择完毕后卡牌飞入建筑详情入口，并在场景内进行建筑建设。然后再出现第二组卡牌选择以此类推重复3次。

**方案2：**先进行选牌，3次选完之后直接进入建筑结算阶段，在结算阶段进行建筑建造。

![](https://cdn.nlark.com/yuque/0/2024/png/43554293/1722912125058-45e9cef4-f583-4037-a552-af99f5a5cdab.png)

#### 建造结算阶段
**方案1：**单纯的结算阶段，无特殊表现。

**方案2：**包含建造建设表现。

#### **地图场景**
+ 场景中设置玩家主城以及默认的攻击武器（红色区域）。
+ 预设场景中划分四个区域对应当前四种资源：工业区、住宅区、科研区、商业区（黄色区域）。

### 战斗表现
#### 攻击阶段使用攻防卡牌
+ 玩家选择攻防卡牌进行打出，直接打出作用自身的卡牌，对其他玩家生效的则拖动至其他玩家头像打出。

![](https://cdn.nlark.com/yuque/0/2024/png/43554293/1722916470199-9892a156-58ec-4b41-9fb3-c323c6d5bbf8.png)

#### 攻击
+ 进行攻击时，点击攻击按钮，弹出攻击选项，选项也类似选择建筑时，弹出4张卡牌对应4类资源，卡牌上标注此次攻击对应的伤害，底部添加倍数选择。
+ 玩家选择后，将对应卡牌拖动至所需攻击玩家的头像上。
+ 系统会随机选择该资源下对应的建筑，根据伤害大小，选择建筑数量会变化。

![](https://cdn.nlark.com/yuque/0/2024/png/43554293/1722937573575-b88d4af1-0313-4e54-8997-fb64dd9971ab.png)

#### 攻击表现
+ 玩家结束攻击阶段时，进入攻击结算，此处地图上展示我方场景中攻击武器发射的动画表现。
+ 根据攻击方式的不同播放不同的攻击动画，但最终都是从我方场景飞往目标场景。
+ 若为2个不同目标则依次进行。
+ 武器直接在我方发射，切换至敌方场景后再播放命中动画。
+ 被直接攻击的建筑变为废墟，若有护盾则展示护盾抵挡效果，根据包装决定表现（比如导弹拦截，电磁护盾等等）。

![](https://cdn.nlark.com/yuque/0/2024/jpeg/43554293/1722920279981-0e603085-e816-4bd8-930f-7a040a4dd9d5.jpeg)![](https://cdn.nlark.com/yuque/0/2024/png/43554293/1722923616603-ffb61714-4cfc-4d68-abf0-77ceea7a90a6.png)

![](https://cdn.nlark.com/yuque/0/2024/jpeg/43554293/1722922434020-7b57a5b9-eb54-4245-a002-f9643e12bc39.jpeg)

#### 攻击结算
+ 正常攻击后若扣血，则主城有损伤表现。
+ 若血量变为0，则主城有破坏表现。
