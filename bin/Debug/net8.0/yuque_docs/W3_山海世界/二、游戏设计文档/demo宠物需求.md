## 1、主要应用场景
注意需要考虑宠物主要出现的场景的视角，以及多个宠物出现在同样位置时整体的和谐性

比较主要的界面有2个

1、战斗界面：战斗界面会有己方背对镜头和敌方正对镜头（类似宝可梦）

2、养成：养成会有一个中心的展示区域
