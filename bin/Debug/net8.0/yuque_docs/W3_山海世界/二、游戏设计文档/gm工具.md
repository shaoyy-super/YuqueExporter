## 3.道具部分
核心部分：增加和删减道具，包括道具表中的全部道具，如果输入错误需要弹出提示“道具不存在”。

增加道具：add，输入道具id，数量

删除道具：delete，输入道具id，数量



快捷按钮： 

宠物球大礼包：直接获得并打开一个奖励包

养成道具大礼包：直接获得并打开一个奖励包

货币大礼包：直接获得并打开一个奖励包

清空背包：删除背包中的全部道具。
