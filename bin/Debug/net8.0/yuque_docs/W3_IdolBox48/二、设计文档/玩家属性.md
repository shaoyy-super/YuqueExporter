# 1、属性类型
## ①三大基础属性
人气、魅力、才艺（包装暂定）

## ②六个二级属性
颜值、身材、气质、粉丝数、曝光度、影响力、舞蹈、唱歌（包装暂定）

## ③三个宝石属性
魅力值、梦幻值、神秘值（包装暂定）

# 2、属性计算

属性4=属性1*（1+属性2）*（1+属性3）

属性14=属性11*（1+属性12）*（1+属性13）
