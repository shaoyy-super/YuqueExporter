# 用户体验
+ 主界面：给予玩家查看解锁偶像卡获得治理币奖励的强展示，刺激玩家购买更多盲盒获得偶像卡
+ 偶像卡界面：给予查看奖励的汇总界面
+ 偶像卡详情界面：着重展示下一级奖励，鼓励玩家尽快升级拿到奖励
