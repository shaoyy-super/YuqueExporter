## 1.3 设计思路
1、为了让玩家更好地体验剧情，在完成难度上做区分

2、通过设计主线系统解锁和完成方式的多样性，不止是让玩家打歌，还可以让玩家体会游戏里的其他功能，尤其是引导到UGC创作上，让玩家领略游戏中的其他功能

3、在交互上尽量让玩家减少操作，在一个页面完成切换
