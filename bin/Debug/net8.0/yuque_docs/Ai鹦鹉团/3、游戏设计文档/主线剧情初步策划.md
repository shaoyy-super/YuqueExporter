## 二、**主线剧情初步策划案**
### 2.1主线剧情模式
主线剧情预计设计为可独立阅读的单元剧，每一小节之间或结尾穿插主线剧情，每一大章节的最后进入主线剧情，推动主线发展。

### 2.2主线剧情概述
#### 制作人身份
1. <font style="background-color:#FBDE28;">【主人公】</font>是曾经的传奇制作人，一手捧出世纪偶像A，但偶像A在出道三年如日中天之时遭遇意外，尸体烧焦/撞烂/不知所踪，在此之后，<font style="background-color:#FBDE28;">【主人公】</font>也消失在公共视野。
2. A是<font style="background-color:#FBDE28;">【主人公】</font>的姐姐，两人父母早逝，缺少亲情之爱的两人相依为命。A和<font style="background-color:#FBDE28;">【主人公】</font>从小在电视和全息投影上被各类偶像陪伴，于是决心要进入偶像行业。A更是想要成为对爱感受颇深/发誓要将爱带给世界的偶像。
3. <font style="background-color:#FBDE28;">【主人公】</font>相比于成为偶像，更想成为制作人。因为发掘偶像的制作人和偶像，正如矿工与金矿，如果没有制作人的发掘，偶像们或许就在自己的生命里度过一生。但因为有了制作人，偶像们的偶像生命才会就此开始。
4. <font style="background-color:#FBDE28;">【主人公】</font>因此向成为创造命运的制作人。



#### 整体设置
##### 明线：
主人公作为大神退隐，来到名不见经传的小工作室，整顿濒临倒闭的小工作室，一步步招募偶像（be like：宝可梦）走向巅峰。

##### 暗线：
追查主人公姐姐的死亡真相，收服【一开始的明面boss】，了解姐姐死亡事件背后的阴谋和姐姐真正的下落。

##### 一句话主线：
主人公的姐姐因主人公青梅竹马的一些操作而死/失踪。

业已成名的主人公从公众视野隐退，颓废一段时间后，决定复出。

这次，主人公既要夺回本属于自己的一切荣耀。

也要将害死姐姐的青梅竹马踩在脚下。

最后还要调查姐姐死亡的真相。

##### 故事简略大纲：
###### 1~4章：【完成偶像的初步收服，组建一个女团，完成第一次中小型比赛，了解女团成员基本性格，透露暗线线索。】
+ 主人公为隐退的天才制作人，在一次偶遇中，帮助了面临工作室危机的【女主角-1】。成功度过危机后，【女主角-1】表示工作室是自己妈妈的梦想，希望主人公可以带领整个工作室走向成功。主人公因有自己的打算，故而选择加入了【女主角-1】的工作室，并和【女主角-1】一起为工作室重新命名。
+ 由于工作室目前只有【女主角-1】一个艺人，为了让工作室打出知名度，主人公安排【女主角-1】去参加商演/比赛/路演等等，在这个过程中，主人公认识了更多的艺人【角色】，他们或主动或被邀请，加入了主人公的工作室。工作室的知名度逐渐打响。
+ 在主人公他们工作室发展期间，有大公司想要仗势欺人用手段收购主人公公司/或主人公公司遇到其他危机，主人公带着女团参加比赛/制作公演，获得名声，抵抗了危机。

###### 5~8章：【完成偶像的进一步收服，组建一个男团，完成一次大型比赛，透露更多暗线线索。】
+ 主人公继续通过发掘地下室偶像团/星探/招募/大公司贵公子陨落等等途径组建男团，并通过参加某次活动赢得奖项，进一步打响名声。
+ 同时，在以上故事推进过程中，主人公见到了故人——【敌对角色男-1】，并进行一些故事闪回，暗示主人公曾经有一段辉煌的过去，而主人公姐姐的死正是与【敌对角色男-1】有关。

###### 9~12章：【培养角色好感，带领团队参加超大型总选，并取得胜利。同时揭露敌对角色男的身份。】
+ 随着故事的进行，X年一度的偶像总选大会快要来临，主人公决定带团队参加，并鼓励大家。在选举会上，最后厮杀的两个团就是主人公的团和【敌对角色男-1】的团，【敌对角色男-1】既是制作人，也是团队成员。最后主人公的团险胜对方。

###### 13~16章【收服敌方角色，揭开更多暗线谜底，进入部分大主线，招募新人——新人可设置为真正的幕后boss方的人。】
+ 由于【敌对角色男-1】的团在最后的选举前签署了高昂的对赌协议，现在对赌失败，面临高额的赔偿，【敌对角色男-1】的团濒临解散。关键时刻，主角伸出橄榄枝，表示自己的工作室/公司可以运营【敌对角色男-1】的团。【敌对角色男-1】很警惕，直言主角别有用心。主角也并不掩饰，表示自己能找打机会把【敌对角色男-1】踩在脚下，当然会开心。随后主角为【敌对角色男-1】阐述了利弊。表示这是他牺牲自我保全队友的唯一机会。【敌对角色男-1】最后答应了主人公开出的条件。
+ 故事继续发展，在此期间，又有新人加入了主角的工作室。主人公在追查姐姐当年死亡真相时，落入险境，时【敌对角色男-1】和其他偶像联合救出了主人公。经过此事，主人公质问【敌对角色男-1】姐姐的死是否另有隐情。【敌对角色男-1】不愿多说。主人公此时已经明白，姐姐的死虽然与【敌对角色男-1】有关，但他并非主谋。

###### 之后的章节安排【工作室走向成熟化，解决姐姐死亡的谜团，引出新的谜团。】
+ 接下来主人公会继续追查姐姐死亡的真相，后续会牵扯出譬如大公司打压偶像，离间主人公和【敌对角色男-1】/科技公司为了宇宙偶像计划让姐姐假死，目的时为了提取姐姐的信息让姐姐作为复活的仿生人出道/末世城邦之间更黑暗的计划，通过击垮主角所在的城邦人们的精神偶像，从而让士气低迷，趁机进行政治活动……诸如此类的发展。

#### 故事开头引入思路
##### 思路阐述：
+ 为主角配置一个固定的秘书类角色，为角色提供引导和陪伴功能。
+ 为起名系统/公司命名系统/等新手指引系统做包装。
+ 简单呈现主人公性格与定位。

##### 思路演示【非定稿】：
+ 故事引入，<font style="background-color:#D9EAFC;">根据选择的性别——【性别系统】</font>，小姑娘/男生问自己的姐姐手里的小卡是谁？姐姐遗憾表示，这可是过去当红的偶像，想不到你已经不认识了。
+ 一阵风吹起，姐姐的小卡掉在地上，主角的手探入，捡起小卡，愣了一会儿，还给姐姐，姐姐道谢，主角只是淡淡应答，绕过姐姐。
+ 【以上剧情之前，主角暂时不露脸】
+ <font style="background-color:#FBDE28;">【主人公】</font>推门而入，进入【公司】，【角色1-女】出现，和【npc-男】正忙得不可开交，见到主角之后，两人都愣住，<font style="background-color:#FBDE28;">【主人公】</font>尴尬一笑，表示，自己只是来借个洗手间。
+ 【角色1-女】和npc失望。主角从洗手间出来后，【npc-男】与【角色1-女】发生对话，大意为：这是【女角色-1】的工作室，工作室为好不容易争取来的校庆演出准备了很久，没想到工作室旗下唯一的偶像却违约签了出价更高的大公司，临阵脱逃。如果这次校庆演出无法出席，工作室会面临赔款，这将成为压垮工作室的最后一根稻草。
+ 【女角色-1】失魂落魄跌坐在地，表示难道就到这里了吗？这是妈妈留给自己的工作室，也是自己的梦想，自己还不想现在就放弃。
+ 【npc-男】愁眉苦脸表示可是……我们能做的都做了，天要亡我，非战之罪……
+ 【女角色-1】站起来擦干眼泪，表示不到最后一刻自己是不会放弃的，没有偶像，大不了我自己上！
+ 【npc-男】震惊，说，怎么可能？！
+ <font style="background-color:#FBDE28;">【主人公】</font>的声音此刻插入，说，怎么不可能。
+ 【女角色-1】和【npc-男】齐刷刷转过头朝<font style="background-color:#FBDE28;">【主人公】</font>。
+ 主角指了指工作室的标语，表示这可是你们自己说的，成为偶像的第一步，就是有一颗想成为偶像的心。
+ 随后主角开始评估【女角色-1】的状况，表示她的身体条件/身体协调程度/外貌/声音去做偶像是够格的，校庆表演的话，稍加训练没问题的。
+ 【女角色-1】凑上来，直接问，你叫什么名字？主角犹豫了一下，说，叫我——<font style="background-color:#D9EAFC;">【起名系统】</font>——就好，【女角色-1】表示，我看出来了，你对偶像演出很有经验？如果现在训练我，能让我赶上校庆吗？
+ <font style="background-color:#FBDE28;">【主人公】</font>一愣，推脱说自己的只是路过。【女角色-1】继续恳求<font style="background-color:#FBDE28;">【主人公】</font>，<font style="background-color:#FBDE28;">【主人公】</font>答应，并在【新手引导】下做出种种游戏操作，帮助【女角色-1】赢得胜利。
+ 事后，<font style="background-color:#FBDE28;">【主人公】</font>了解了这间工作室对【女角色-1】的意义，在【女角色-1】的诚挚邀请下，加入了工作室。【女角色-1】因对<font style="background-color:#FBDE28;">【主人公】</font>心怀感激，希望工作室重新整顿，并邀请<font style="background-color:#FBDE28;">【主人公】</font>为工作室命名。
+ <font style="background-color:#FBDE28;">【主人公】</font>命名工作室——<font style="background-color:#D9EAFC;">【工作室起名系统】。</font>
