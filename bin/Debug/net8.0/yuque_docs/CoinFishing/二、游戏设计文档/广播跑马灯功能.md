## **设计思路**
+ 通过全服广播发送重要系统通知及玩家动态，满足系统通知功能及玩家炫耀需求

# **功能说明**
## 全服广播
+ 广播队列无上限
+ 广播队列区分优先级，高优先级可插队播放
+ 广播表现：
    - 每条广播消息从左往右轮播
    - 每条广播有固定播放次数：首字从左侧出现到最后一个字在右侧消失算播一次
    - 多条广播消息间间隔2s【暂定】
    - 广播消息轮播速度根据文字长短支持配置
    - 广播消息支持富文本
+ 广播跑马灯会出现在全服频道
