# 二、功能说明
## 2.1 功能设定
a、服务器开始时默认开启，入口默认存在红点。

b、点击入口打开问卷调查界面，跳转外部链接（由运营提供）

c、完成问卷调查，入口消失。奖励通过邮件形式发放
