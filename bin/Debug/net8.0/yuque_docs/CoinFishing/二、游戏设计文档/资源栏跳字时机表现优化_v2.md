+ 点击炮台发射时，数字立即扣除
+ **普通鱼**：捕获到鱼时，增加金币数，同时向上冒出“+获得金币数”
    - 若连续捕获多条鱼，后面的跳字打断前面的跳字，冒出跳字的速度加快
+ **boss鱼**：在boss表现播完后播金币跳字
