## **设计思路**
+ 弱引导开放式教学-教学核心逻辑：炮倍升级+金币消耗
+ 引导阶段必要功能简化操作
+ 把控奖励与刺激节奏，不断给予玩家爽感，逐步提升玩家鱼倍与金币储备到达预设定数值。
+ 美女荷官人声引导，营造赌场游戏感

# 2 功能说明
### 2.1 引导全流程
#### 2.1.1 框架流程
1. 新手建号：玩家创建账号，进行身份相关的验证
2. 直接进入渔场，渔场为定制关卡，允许脱离，脱离后再进入为正常初级场。

![](https://cdn.nlark.com/yuque/0/2024/png/45603655/1732527084448-04ad6071-a7f8-4090-a27d-204c60667189.png)

3. 特定新手关：定制boss类型（持续留场boss），定制新手鱼群。
+ 定制boss——两类boss：一类是站场高击中BOSS，可以不断的攻击获取奖励的（限定1只），新手场提高击中奖励概率（鱼乐的祥瑞福狮）；一类是美人鱼这种击杀BOSS，新手村提高击杀概率（限定1只）。
+ 新手鱼群：进入渔场前x分钟的鱼群都有一定概率的击中提升。
+ 引导女声——以半身像+头像的形式，配合声音，引导玩家操作，例如：哥哥这里，老板点下这里，哇塞老板好棒！
4. 递进奖励任务-不简短的升级奖励：
+ 从开始大概5秒钟给予1次任务与奖励反馈，在前10分钟内可以设置连绵不断的任务，玩家可以不断的领取金钱和道具。
5. 功能引导：弱引导的UI去强调使用某些功能，且玩家都只需要点击一下就可以生效，无需进入多层ui。
6. 非强制性的渔场转换，炮台转换，回归玩家的正常节奏。
7. 定制UI，新手场地ui与初级场不同，定制部分ui，例如渔场内没有活动按钮，没有各类商品入口按钮。
