## 三、冻结定义
1 冻结目前定义为使用后，冻结使用时在场景内所有处于可击中状态鱼类。

不包含：

+ 在不可击中状态下的入场表现
+ 鱼类死亡后表现
+ UI与镜头层面的表现（包含部分鱼类身上的特殊UI）
+ 场景效果
+ 特效层面表现

2 冻结在使用后会有相对应的场景效果表现

3 冻结道具具备冷却cd。
