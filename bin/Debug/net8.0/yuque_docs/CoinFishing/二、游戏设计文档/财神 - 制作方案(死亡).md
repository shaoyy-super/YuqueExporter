# 1、TML拆分
1）全局TML（1条）

负责：结算入场+跳字+事件帧+结算退场

美术制作：

+ 入场表演
+ 结束退场
+ 调起跳字UI
+ 跳字整合+消失
+ 衔接事件帧

程序控制：

+ 结算开始就播放这个TML
+ 等整个结算结束后停止
+ 停止时机：整个过程结束（开奖失败，或者大奖结束）



2）开奖蓄力（1条）

负责：进口袋掏的过程

美术制作：

+ 进口袋掏的动作
+ 越来越强的光效
+ 衔接事件帧



3）开奖（3条，小奖、鱼-大奖、龙-大奖）

美术制作：

+ 不同挡位的开奖表演
+ 跳字控制
+ 元宝、鱼、金龙等TML自己创建
+ 衔接事件帧

程序控制

+ 根据结果填充跳字参数
