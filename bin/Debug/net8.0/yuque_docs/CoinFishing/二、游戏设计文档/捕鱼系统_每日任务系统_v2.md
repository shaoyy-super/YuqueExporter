### 3.2 每日任务规则
1、每日任务是非线性的，可以同时完成。

2、每日任务0点刷新。

3、若有多个奖励都完成了，点击则一次性全部领取。

4、任务排序再id排序的基础上，按照已完成未领取→未完成→已完成已领取。
