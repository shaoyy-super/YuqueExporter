1.注册监听UI选择事件

2.根据监听事件返回数字重新排列actor

3.为选择的actor上的3DUI赋值数字

额外事件:

TmlGroup:4:处理时间截止时随机选择卡牌

TmlGroup:5:重新排列Actor数组
