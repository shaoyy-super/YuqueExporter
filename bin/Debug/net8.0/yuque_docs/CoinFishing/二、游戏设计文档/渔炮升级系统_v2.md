## **设计思路**
+ 给渔炮添加数值养成线，消耗材料给渔炮升级，获得更高级的加成效果

# **功能说明**
玩家消耗材料来解锁更高的渔炮炮倍。

**锻造成功率：**

+ 每次锻造有成功率，失败时返还部分材料
+ 有初始成功率，每次失败后，下一次成功率+x%
+ 成功后下一次锻造成功率重置为初始成功率

**升级材料来源**：待定。
