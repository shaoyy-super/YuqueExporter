## 简介：
用于隐藏**当前场景根目录下**的**Scene_Transition**节点。

## 参数细节介绍
### 1.轨道参数
无

### 2.片段参数
![](https://cdn.nlark.com/yuque/0/2024/png/22817384/1713943194457-d586a72a-8edf-467a-a031-6b9a3e1d54ed.png)

点击想要调整的片段，显示右边的片段参数

| **参数** | **说明** |
| :--- | :--- |
| 开启黑屏渐变 | √：隐藏场景前播放一个黑屏渐变，在最黑的时间点隐藏场景；同时退出clip前也会播放一个黑屏渐变，在最黑的时间点还原场景<br/>×：进入clip时隐藏场景，离开clip时还原场景 |
| 渐变时长 | 渐变的时长，大于clip时长会报错。为0时<br/>无渐变。 |


为了方便理解，可参考下图时间轴：

![](https://cdn.nlark.com/yuque/0/2024/png/22817384/1713943194652-19a49fe1-588f-4e75-9b92-c9a1590a8264.png)

假如我们勾选了**开启黑屏渐变**，且**渐变时长为2s**，则上图时间点的行为是：

A：开始变黑

B：屏幕全黑（此时会隐藏场景），黑屏开始逐渐变淡

C：黑屏消失

C-D：无黑屏，场景处于隐藏状态

D：开始变黑

E：屏幕全黑（此时会还原场景），黑屏开始逐渐变淡

F：黑屏消失
