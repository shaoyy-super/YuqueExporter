### 面数检查(需选中)
检查选中的模型集合面数密度是否超过设定值

![](https://cdn.nlark.com/yuque/0/2024/png/22817384/1715858206129-e99bba40-378e-4acb-b893-c7af3e9b929c.png)

每单位面积面数(面数密度) = 模型整体三角面数 / 最大面的面积

最大面的面积:选取模型包围盒最大面的面积

检查可以对选中的模型集合分别检查或者将模型集合看做一个整体
