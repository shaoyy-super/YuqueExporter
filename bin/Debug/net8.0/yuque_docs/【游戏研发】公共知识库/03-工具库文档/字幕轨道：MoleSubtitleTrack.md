| :--- | :--- |
| 用途简介 | 用于显示字幕，字幕的样式可以自定义，字幕的内容读取stringId以适配多语言 |
| 融合说明 | 无融合 |
| Track/Clip 结束说明 | 无特殊处理 |
| 要点描述 | 在非运行模式下使用该轨道时，需要保证场景中能找到UICanvas/GUIRoot/UI_Effects_Panel。同时，非运行模式下无法读取stringId，因此会显示错误的文字。<br/>如果要看正确效果，推荐在运行时进行编辑和查看。 |
