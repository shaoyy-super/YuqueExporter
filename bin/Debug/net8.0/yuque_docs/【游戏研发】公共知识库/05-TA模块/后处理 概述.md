# **概述**


**后处理包含：**SSAO, 体积光（VolumetricLight）,景深

**SSAO参数:** -1开 -2关

**体积光参数:** -1开 -2关

**景深参数(高斯模式)：**

-1开，-2关

模糊次数 3，5

| 高 | SSAO -1,VolumetricLight -1,DepthOfField -1，5 | 超精细 |
| :--- | :--- | :--- |
| 中 | SSAO -2,VolumetricLight -1,DepthOfField -1，3 | 精细 |
| 低 | SSAO -2,VolumetricLight -2,DepthOfField -1，3 | 普通 |
| 关闭 | SSAO -2,VolumetricLight -2,DepthOfField -2，3 | 流畅 |
