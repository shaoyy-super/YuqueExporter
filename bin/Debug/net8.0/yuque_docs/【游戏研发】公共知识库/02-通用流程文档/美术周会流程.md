## 周会目的
1. 确认美术进展是否符合预期
2. 确认任务优先级是否需要调整
3. 检查是否有遗漏需求
4. 确定当周关注的重点任务
5. 确定哪些任务不能延期
6. 人力饱和度讨论
7. 问题收集和讨论

## 参会人员
+ 丁丁、策划根据当周情况参与
+ PM：项目PM、美术PM（主持人）
+ 美术leader：童、徐鹏、joy
