## 用途：
1. 用于在游戏内显示头像，有点击回调。
2. 使用场景：
    1. 需要显示头像的地方
3. 扩展：
    1. 可以添加更多的显示信息

## 用法：
1. 预制体路径：Assets/Art/ModuleRes/Item/UI_Head_Item.prefab![](https://cdn.nlark.com/yuque/0/2024/png/35004992/1726021816042-b019e95a-237c-41cb-be4a-9564c6033f13.png)
2. Lua脚本路径：Presentation/CSys/Items/UI_Head_Icon.lua
3. 使用方法：
    1. 加载预制体后，通过New方法创建UI_Head_Icon对象，第一个参数为预制体，第二个为SpriteRegistry，用于加载头像图标。![](https://cdn.nlark.com/yuque/0/2024/png/35004992/1726022289141-768f5947-715b-48ce-af4e-12bedf4a5455.png)
    2. 调用UI_Head_Icon的Show方法，第一个参数为头像名，第二个参数为点击回调，可以不传值。

![](https://cdn.nlark.com/yuque/0/2024/png/35004992/1726022343688-8e5c23d8-56e1-4e1b-8002-eec6ad453537.png)

    3. 调用UI_Head_Icon的SetSize方法，设置头像大小，参数为高度、宽度，默认为100x100![](https://cdn.nlark.com/yuque/0/2024/png/35004992/1726022436594-c397e645-cf92-4563-bf5c-a6521fd5630b.png)
    4. 使用完毕后调用Release方法释放资源

![](https://cdn.nlark.com/yuque/0/2024/png/35004992/1726022449334-4b72daf9-b905-40b3-87fe-1a2d81eecd0c.png)
