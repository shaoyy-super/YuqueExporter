# 异常情况:
### 点击之后未创建prefab
![](https://cdn.nlark.com/yuque/0/2024/png/43288772/1712826871360-23828bb2-644f-4579-a1be-a47b85baccde.png)

解决办法: 查看控制台是否有此log, 有的话先在prefab里点一个组件, 再次创建, prefab会创建在点击的组件下, 其他情况下的报错找对应模板的开发者处理.
