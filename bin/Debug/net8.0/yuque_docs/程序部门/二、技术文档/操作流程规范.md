## 准则：
保证线上可用且正确为第一准则，即不可拿线上环境做测试以及临时操作

## 操作流程：
### 1. 标准化操作流程
现有已制定的标准化操作流程，如新区上线、新机上线等。标准化操作流程需要有对应自动化的工具、流程操作文档以及CheckList。
