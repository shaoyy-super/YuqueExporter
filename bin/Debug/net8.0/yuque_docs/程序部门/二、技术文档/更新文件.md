### 更新文件
+ 使用root登陆目录服，将需要更新的文件上传至/data/salt/states/game/files/templates目录下，此目录下的结构对应游戏服目录的结构/data/zones/xxxx。

![](https://cdn.nlark.com/yuque/0/2024/png/43288467/1713176459643-f268baca-bf55-4185-a19d-948e9ed14d97.png)

+ 在目录服上执行命令

salt '*' state.apply game.upgrade pillar='{"zone": "*", "env": "all"}'

上述命令将递归更新/data/zones/xxxx/目录下的

1. config目录
2. db目录
3. start-one.sh 脚本
4. start.sh 脚本
5. stop.sh 脚本

**注意****<u>：不要更新config目录下的config.xml，避免导致合服信息的丢失。</u>**
