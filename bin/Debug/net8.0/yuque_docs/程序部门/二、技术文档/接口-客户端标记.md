### 目的
用于客户端标记设置，能自动同步到服务器存储

### 接口
--

-- brief 获取变量标记

-- param player 玩家对象

-- param tag 标志类型

-- return 标志值（true or false)

--

PlayerSetting:GetTag(player, tag)--

-- brief 设置变量标记

-- param player 玩家对象

-- param tag 标志类型

-- param value 标志值（true or false)

--

PlayerSetting:SetTag(player, tag, value)
