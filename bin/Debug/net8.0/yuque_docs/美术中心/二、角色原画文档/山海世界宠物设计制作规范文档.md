## **<font style="color:rgb(0,0,0);">提交审核流程与审核重点：</font>**
1.初步草稿方案提交审核（确定剪影，写实程度）

2.确定方案进行优化处理（设计逻辑闭环处理）

3.进行三视图制作（明确各个角度制作细节）

4.排版整合提交审核（排版格式见上方说明）

5.上传SVN归档
