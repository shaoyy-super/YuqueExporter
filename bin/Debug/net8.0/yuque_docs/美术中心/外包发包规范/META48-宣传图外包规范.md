## **流程与规范：**
输出格式：输出分辨率300，尺寸3840×2160。角色与背景分离，如有特效，气氛需分层。

流程：构图草图（2个左右）→色稿草图→细化完成度60%→提交细化完成度100% 。

构图草图阶段:
