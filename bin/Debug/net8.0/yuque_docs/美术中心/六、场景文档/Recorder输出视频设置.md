## N卡自带录制工具
![](https://cdn.nlark.com/yuque/0/2024/png/44842280/1717671835984-31f89076-c02a-4cd7-af7d-fa37d5166598.png)

1.如果电脑没有安装，需要先安装N卡驱动程序

2.登录账号进入GeForce experence

3.快捷键alt+z，唤醒GeForce，对录制功能进行设置

4.快捷键alt+F9开始（停止录制）
